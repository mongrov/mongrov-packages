# Changelog

All notable changes to `@mongrov/theme` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - 2026-04-03

### Added

- Comprehensive test coverage for theme functionality
- Additional color scheme utilities
- Improved TypeScript types

### Changed

- Enhanced theme provider performance
- Better dark mode detection
- Improved documentation

## [0.2.0] - 2025-12-01

### Added

- `ColorSchemeStore` for persistent color scheme preferences
- System color scheme detection
- `useColorSchemeStore` hook

### Changed

- Improved theme switching performance
- Better SSR support

## [0.1.0] - 2025-10-01

### Added

- Initial release of `@mongrov/theme`
- **Theme System**:
  - `createTheme` factory function
  - Color scheme definitions (light/dark)
  - Theme contract types
- **React Integration**:
  - `ThemeProvider` context
  - `useTheme` hook
  - `useColorScheme` hook
